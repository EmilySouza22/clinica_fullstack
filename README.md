## Sobre o Projeto
Esse projeto é uma aplicação web fullstack para gestão de clínicas, permitindo cadastro de pacientes, agendamento de consultas, controle de prontuários e acompanhamento financeiro (receitas de consultas/procedimentos) em tempo real, com histórico completo de atendimentos. O objetivo é centralizar dados clínicos e financeiros em uma única plataforma, com backend em Node.js/TypeScript, ORM Prisma para persistência em PostgreSQL, frontend em React, e análise de dados para relatórios de ocupação, receita e frequência de pacientes.
